# Contributing to DemocracyFlow

We welcome contributions to DemocracyFlow! Our mission is to make the voting process simple, secure, and accessible for everyone.

## Getting Started
1. Fork the repository
2. Run `npm install`
3. Run `npm run dev`

## Guidelines
- Write tests for new features.
- Ensure 0 `any` types in TypeScript.
- Follow our `.prettierrc` formatting.
